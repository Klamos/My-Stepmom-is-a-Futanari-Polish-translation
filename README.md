# My-Stepmom-is-a-Futanari-Polish-translation
Spolszczenie do gry My Stepmom is a Futanari by Michał Klamer

Oryginalna treść dostępna pod adresem: https://steamcommunity.com/sharedfiles/filedetails/?id=2565346634

Instalacja
Aby zainstalować spolszczenie, należy pobrać PLIKI i wypakować je w głównym folderze gry.
( D:\Program Files\Steam\steamapps\common\My Stepmom is a Futanari ).
Najszybciej zrobisz to klikając prawym przyciskiem na grze w bibliotece Steam ➔ Zarządzaj ➔ Przeglądaj pliki lokalne.
W każdej chwili masz możliwość powrotu do języka angielskiego, poprzez wybranie opcji English w menu.
